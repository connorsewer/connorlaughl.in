# Links Should Carry Semantic Meaning

A bare link is a pointer. A link in prose is guidance.

Prefer:

- "Use [[progressive-disclosure-is-a-capability-multiplier]] when designing retrieval-heavy docs."

Over:

- "See also: progressive-disclosure-is-a-capability-multiplier"

The sentence teaches the traversal rule.
