# Sanity CMS Setup for connorlaughl.in

## Quick Start

```bash
cd /Users/connorlaughlin/clawd/career/portfolio-v2/sanity
npm install
npx sanity@latest init --coupon crockford2023
```

## After Setup

1. Copy the project ID into `sanity.config.ts`
2. Deploy with `npx sanity deploy`
3. Access your CMS at: https://your-project.sanity.studio

## Schema Types

- **Case Study**: Full case studies with proof points, outcomes, governance
- **Homepage Hero**: Edit the main headline, stats, CTAs
- **Page Content**: Flexible content blocks for any page
