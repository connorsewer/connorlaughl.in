# Obsidian Is a Good Interface, Not a Requirement

Obsidian makes linked markdown pleasant for humans.

What matters more than the app:

- plain markdown files
- stable folder structure
- meaningful titles
- descriptions/frontmatter
- linkable atomic notes
- reliable retrieval habits

Use Obsidian if it improves browsing, editing, and stewardship. Do not make the system depend on proprietary behavior.
