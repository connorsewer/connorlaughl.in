# Handoffs

Use this folder for structured cross-session handoff notes.

## When to create a handoff

Create a handoff when:
- work is long-running
- another session/subagent/cron may need to continue it
- there are meaningful next actions or blockers
- a restart or interruption would otherwise lose useful context

## Suggested format

- Context
- What changed
- Artifacts
- Next actions
- Risks/blockers

## Rule

Prefer short, high-signal handoffs over giant session dumps.
