# Project Handoff Template

## Context
- What is this project/task?
- Why does it matter right now?

## Current state
- What has been done?
- What is in progress?

## Artifacts
- Relevant files, notes, links, or outputs

## Next actions
- Immediate next steps
- Ordered if helpful

## Risks / blockers
- What could stall progress?
- What assumptions may be wrong?

## Notes for next session/agent
- Any nuance worth preserving
