# Portfolio v2 (Next.js)

Luxury editorial, dark-first, black/white with rattan accent.

## Run locally

```bash
cd career/portfolio-v2
npm run dev
```

Then open: http://localhost:3000

## Pages
- `/` landing
- `/proof` interactive proof explorer
- `/case-studies/[slug]` case study template
- `/case-studies/strategy-memo` narrative anchor
