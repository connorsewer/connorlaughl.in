# Operationalize Repeated Work

Repeated work should become easier over time.

## Promotion ladder

- one-off insight → daily memory
- repeated pattern → graph node
- repeated execution sequence → runbook or workflow note
- timed recurring task → cron or heartbeat checklist
- stable specialized capability → skill
