# Progressive Disclosure Is a Capability Multiplier

The best retrieval decision often happens before a full file read.

A good graph supports this sequence:

1. Start at an index or MOC
2. Read short descriptions and note titles
3. Follow only the links that match the task
4. Read full node content only when needed

That preserves attention for reasoning and execution.
