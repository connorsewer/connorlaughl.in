# Layered Memory Beats Bigger Context

Large context helps scanning, but durable performance comes from storing the right information in the right layer.

## Layers in this workspace

- Orientation/session memory: `SOUL.md`, `USER.md`, recent daily notes
- Curated long-term memory: `MEMORY.md`
- Structured reusable knowledge: `graph/`
- Operational traces and drafts: `ops/`

This makes it easier to preserve continuity without loading everything every time.
