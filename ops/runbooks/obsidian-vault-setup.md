# Obsidian Vault Setup

## Recommendation

Open `/Users/connorlaughlin/clawd` itself as the Obsidian vault.

## Why

- same files for human and agent
- markdown remains canonical
- git history stays attached to the real workspace
- no sync drift between repo and vault

## Primary folders

- `MEMORY.md`
- `memory/`
- `graph/`
- `ops/runbooks/`
- `ops/tmp/`
- `ops/drafts/`

## Minimal setup

1. Open the repo as a vault in Obsidian
2. Verify wiki links resolve cleanly
3. Enable backlinks and graph view if desired
4. Keep plugins minimal at first

## Plugin guidance

Start with core Obsidian only.

Optional later:
- Dataview
- Templates or QuickAdd

## Rule

Obsidian is the interface. The files are the system.
